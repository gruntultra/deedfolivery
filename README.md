# Deed Folivery

Deed Folivery is a food delivery website.

## Installation

Clone this repo.

### Server

1. `cd` into server.
2. Create a psql database called `testdb`
3. Run `\i schema.sql`.
4. Go back to the server directory.
5. Run `yarn install` and `yarn start`.

### Client

1. `cd` into client.
2. Run `yarn install` and `yarn start`.
